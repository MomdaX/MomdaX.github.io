# 中图铁路地图 - 技术文档

## 📋 项目概述

这是一个基于Web的中图铁路地图可视化系统，实现了铁路网络的图形化展示和最短径路查询功能。用户可以通过输入终点站名称或电报码，快速生成并展示从固定起点（钦州港）到目标车站的最短铁路径路。

### 功能3: 最短径路生成 ⭐⭐⭐

```
技术实现：A*算法 + Dijkstra算法
```

#### 3.1 算法选择策略

系统采用**智能算法选择**策略，根据不同情况选择最优算法：

| 场景 | 使用算法 | 原因 |
|------|----------|------|
| 普通车站查找 | **A\*算法** | 启发式搜索，效率高 |
| 考虑里程约束 | **Dijkstra算法** | 保证最短路径 |
| 绕过禁行路段 | **A\* (ignoreNotPass)** | 忽略notPass标记 |

#### 3.2 A\*算法详解

**算法原理：**
```
A* = Dijkstra + 启发式函数
f(n) = g(n) + h(n)

其中：
- g(n): 从起点到当前节点的实际代价
- h(n): 从当前节点到终点的预估代价（启发函数）
- f(n): 综合代价，越小越先被探索
```

**启发函数设计：**
```javascript
// 使用欧几里得距离作为启发函数
heuristic(node1, node2) {
    var dx = node1.x - node2.x;
    var dy = node1.y - node2.y;
    return Math.sqrt(dx * dx + dy * dy) * 0.5;  // 乘以系数平衡
}
```

**A\*算法核心流程：**
```
1. 初始化开放列表和关闭列表
2. 将起点加入开放列表
3. 循环执行：
   a. 从开放列表取出f值最小的节点
   b. 如果是目标节点，重建路径返回
   c. 遍历该节点的所有邻居：
      - 计算g值（经过当前节点的代价）
      - 如果新路径更优，更新父节点和g值
      - 如果邻居不在开放列表，加入
   d. 将当前节点加入关闭列表
4. 如果开放列表为空，未找到路径
```

#### 3.3 Dijkstra算法详解

**算法原理（贪心策略）：**
```
- 使用优先队列（最小堆）管理待探索节点
- 每次取出的都是当前已知最短距离的节点
- 保证找到全局最优解
```

**代码实现：**
```javascript
dijkstra(startNodeId, endNodeId, ignoreNotPass) {
    var gScore = {};          // 存储每个节点的最短距离
    var cameFrom = {};        // 存储路径
    var openSet = [];         // 优先队列
    
    // 初始化所有节点距离为无穷大
    for (var i = 0; i < nodes.length; i++) {
        gScore[nodes[i].node] = Infinity;
    }
    gScore[startNodeId] = 0;
    
    while (openSet.length > 0) {
        // 取出距离最小的节点
        openSet.sort((a, b) => a.cost - b.cost);
        var current = openSet.shift();
        
        if (current.nodeId === endNodeId) {
            return reconstructPath(cameFrom, current.nodeId);
        }
        
        // 遍历邻居，更新距离
        var neighbors = this.getNeighbors(current.nodeId);
        for (var i = 0; i < neighbors.length; i++) {
            var neighbor = neighbors[i];
            var tentativeG = gScore[current.nodeId] + neighbor.distance;
            
            if (tentativeG < gScore[neighbor.nodeId]) {
                cameFrom[neighbor.nodeId] = current.nodeId;
                gScore[neighbor.nodeId] = tentativeG;
                openSet.push({ nodeId: neighbor.nodeId, cost: tentativeG });
            }
        }
    }
    return [];
}
```

#### 3.4 路径优化策略

**多路径对比机制：**
```javascript
// 1. 计算普通路径（考虑禁行）
var normalPath = this.aStar(startNode.node, targetNodeId, false);

// 2. 计算忽略禁行路径
var ignorePath = this.aStar(startNode.node, targetNodeId, true);

// 3. 对比距离，选择最优
var paths = [];
if (normalPath.length > 0) 
    paths.push({path: normalPath, dist: calculateDistance(normalPath)});
if (ignorePath.length > 0) 
    paths.push({path: ignorePath, dist: calculateDistance(ignorePath)});

// 4. 选择最短路径
paths.sort((a, b) => a.dist - b.dist);
path = paths[0].path;
```

### 核心代码实现

#### 1. VBA端代码

```vba
Public ad As Object  ' 全局COM对象引用

Public Function Getadobj(aardio As Object)
    Set ad = aardio
    Debug.Print "[" & Format(Now, "HH:mm:ss") & "]：" & ad.msg
End Function

Sub CallAnything()
    ad.cx "新晃"  ' 测试调用
End Sub

Private Sub Worksheet_BeforeDoubleClick(ByVal Target As Range, Cancel As Boolean)
    Dim val As String
    
    ' 只响应H列（到站列）
    If Target.Column <> 8 Then Exit Sub
    Cancel = True
    
    val = Trim(Target.Value)
    
    If val <> "" Then
        Dim exePath As String, exeName As String
        exeName = "铁路地图SVG.exe"
        exePath = ThisWorkbook.Path & "\" & exeName
    
        ' 进程管理逻辑
        If ad Is Nothing Then
            ' 首次启动：关闭旧进程，启动新程序
            CreateObject("WScript.Shell").Run "taskkill /f /im " & Chr(34) & exeName & Chr(34), 0, True
            CreateObject("WScript.Shell").Run Chr(34) & exePath & Chr(34), 1, False
        Else
            ' 已连接：检查进程状态
            If IsPR(exeName) Then
                ' 进程已关闭，重新启动
                Set ad = Nothing
                CreateObject("WScript.Shell").Run Chr(34) & exePath & Chr(34), 1, False
            Else
                ' 进程正常，直接调用
                ad.cx val
            End If
        End If
    End If
End Sub

Function IsPR(exeName As String) As Boolean
    On Error Resume Next
    With GetObject("winmgmts:\\.\root\cimv2").ExecQuery("select * from Win32_Process where Name='" & exeName & "'")
        IsPR = .Count = 0
    End With
    On Error GoTo 0
End Function
```

```aardio
import win.ui;
import web.view;
import wsock.tcp.simpleHttpServer;
import console;
import com.excel;

var winform = win.form(text="中图铁路地图";right=1200;bottom=800)
winform.add(

var wb = null;

// 启动本地HTTP服务器

// 创建浏览器控件
wb = web.view(winform.custom);
wb.go(httpUrl);

// 初始化Excel与VBA对象
try {
    xl = com.excel(false);
    ad = {
        msg = "连接成功！",
        cx = function(e) {
            if (!wb) return null;
            var cz = "generatePath('" ++ e ++ "')";
            return wb.eval(cz);
        };
    }
    xl.vba.Getadobj(ad);

    console.log("Excel初始化失败:", e);
}

// 窗口关闭事件
winform.onClose = function(sender) {
    // 释放Excel引用
    if (xl) {
        try {
            xl.vba.Getadobj(null);
            xl.Quit();
            com.Release(xl);
        } catch(e) {}
        xl = null;
    }
    ad = null;

    // 关闭浏览器控件
    if (wb) {
        try {
            wb.close();
        } catch(e) {}
        wb = null;
    }

    // 关闭HTTP服务器
    try {
        wsock.tcp.simpleHttpServer.stopUrl();
    } catch(e) {}
    httpUrl = null;

    return true;
}

winform.show();
return winform.wait();
```
