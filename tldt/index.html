算法原理

智能路径规划

融合启发式搜索与贪心策略，找到最优解的黄金组合

A* 算法

启发式搜索
f(n) = g(n) + h(n)
g(n) 从起点到当前节点的实际代价
h(n) 从当前节点到终点的预估代价（启发函数）
1
初始化 将起点加入开放列表
2
选择节点 取出f值最小的节点
3
扩展邻居 计算g值，更新最优路径
4
到达终点 重建路径，返回结果
高效快速
目标导向
性能优越

Dijkstra 算法

贪心策略

优先队列 + 最短距离

优先队列 使用最小堆管理待探索节点
贪心选择 每次取最短距离节点，保证全局最优
初始化 所有节点距离=∞
迭代 取最小 → 更新邻居
完成 找到最短路径
全局最优
稳定可靠
保证准确

VBA代码示例

Private Sub Worksheet_BeforeDoubleClick( _
    ByVal Target As Range, Cancel As Boolean)
    
    ' 只响应H列
    If Target.Column <> 8 Then Exit Sub
    Cancel = True
    
    val = Trim(Target.Value)
    If val <> "" Then
        ad.cx val  ' 调用COM对象生成路径
    End If
End Sub

    msg = "连接成功！",
    cx = function(e) {
        if (!wb) return null;
        var cz = "generatePath('" ++ e ++ "')";
        return wb.eval(cz);
    }
};
