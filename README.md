# Momda_X 网站导航

一个现代化的个人导航主页，集成多种视觉特效和常用网站链接。

## 功能特性

- 🎨 **粒子线条特效** - 交互式粒子背景，支持鼠标跟随
- ❤️ **心形漂浮特效** - 浪漫的心形动画效果
- 📁 **项目导航** - 快速访问调车计划和铁路地图项目
- 🔗 **常用网站导航墙** - 收藏的常用网站快捷访问

## 技术栈

### 前端技术
- **HTML5** - 语义化标签，SEO友好
- **CSS3** - 动画效果、渐变背景、响应式布局
- **JavaScript ES6+** - 模块化开发、异步处理
- **jQuery** - DOM操作、心形漂浮特效

### 核心功能实现

#### 粒子线条特效
- **算法**: 基于 Canvas 2D 的粒子系统
- **实现原理**: 
  - 创建粒子类，包含位置、速度、加速度属性
  - 使用 requestAnimationFrame 实现60fps动画
  - 粒子间距离检测，小于阈值时绘制连接线
  - 鼠标位置监听，计算粒子受力方向

#### 心形漂浮特效
- **SVG路径**: 使用贝塞尔曲线绘制心形
- **动画控制**: 
  - 随机生成心形大小和初始位置
  - CSS transform 实现漂浮和旋转效果
  - 定时器控制动画生命周期

#### 响应式导航
- **Flexbox布局**: 自适应不同屏幕尺寸
- **媒体查询**: 移动端菜单优化
- **平滑滚动**: 页面内锚点导航

### 视觉特效
- **渐变背景**: CSS linear-gradient 动态色彩过渡
- **阴影效果**: box-shadow 增强层次感
- **悬停动画**: 按钮和卡片的交互反馈

### 性能优化
- **节流函数**: 限制鼠标事件触发频率
- **requestAnimationFrame**: 优化动画性能
- **DOM缓存**: 减少重复DOM查询

## 特效说明

1. **粒子线条特效**：白色粒子在背景中随机移动，粒子之间根据距离自动连接形成线条，鼠标移动时粒子会被吸引跟随

2. **心形漂浮特效**：粉红色爱心从页面底部不断向上飘动，大小、位置、颜色随机变化
